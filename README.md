# OSINT-SEARCH
![Bitcoin](https://img.shields.io/badge/bitcoin-2F3134?style=for-the-badge&logo=bitcoin&logoColor=white)
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Recherche des e-mails/users sur différents sites internet.


## Logiciel utilisé

### Prérequis 

- Installer Python ([Installation & Documentation](https://www.python.org/downloads/))


## Installation 

> [!NOTE]
> Il est impératif que toutes les dépendances soient à jour!

`git clone https://github.com/alexcaussades/Osint-Search`

`pip install -r Requirements.txt `

> [!TIP]
> Rendez vous sur le wiki pour une avoir plus d'info

Wiki [installation complète](https://github.com/alexcaussades/Osint-Search/wiki)

### Liste des dépendances obligatoires.

- Maigret ([Installation & Documentation](https://github.com/soxoj/maigret))
    - Maigret collecte un dossier sur une personne uniquement par son nom d’utilisateur, vérifiant les comptes sur un grand nombre de sites (plus de 3K+)

- Sherlock Project ([Installation & Documentation](https://github.com/sherlock-project/sherlock))

    - Recherchez les comptes de réseaux sociaux par nom d’utilisateur sur 400+ réseaux sociaux
